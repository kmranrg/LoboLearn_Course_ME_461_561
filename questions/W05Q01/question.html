Distance to an SVM Hyperplane

In a 2-D Support Vector Machine (SVM) classifier, the decision boundary (hyperplane) is

\( \mathbf{w}^{\!\top}\mathbf{x} + b = 0 \)

The (unsigned) distance from any point \( \mathbf{x}_0 \) to this hyperplane is

\( r = \dfrac{\lvert\,\mathbf{w}^{\!\top}\mathbf{x}_0 + b\,\rvert}{\lVert \mathbf{w}\rVert} \)
